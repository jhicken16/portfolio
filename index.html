Location Collector

This project is part of my boot camp curriculum. The objective is to create an application using the Leaflet JavaScript library with a PHP backend that communicates with other APIs.

The initial call to a third-party API is to OpenCage. This API retrieves general information about the area where the user has clicked, such as the address, and stores it in the main data structures.

To Be Developed: Once a user selects a location, additional options will be available to collect more information using other third-party APIs. These enhancements will enable the addition of more features. After development is complete, users will be able to click on the map or search for a location, select a specific area, and search for nearby points of interest, which will also be displayed on the map. Users can plan routes based on the selected points of interest and reorder their route by adjusting positions within an array.
